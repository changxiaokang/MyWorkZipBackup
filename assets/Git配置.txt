1、设置ID信息

git config --global user.email "changxiaokang@hotmail.com"

2、生成	SSH key
mkdir .ssh	// 创建.ssh目录
ssh-keygen -C "changxiaokang@hotmail.com" -t rsa

Git代码提交：
＝＝＝＝＝＝
1、GitHub建立好仓库

2、下载GitHub工程
git clone https://github.com/changxiaokang/AFK.git
Cloning into 'changxiaokang'...
warning: You appear to have cloned an empty repository.

3、添加代码进来

4、更新内容到索引
git add .

5、查看那些改动将要被提交
git diff --cached

6、提交代码
git commit -m "说明"

7、查看日志
git log
git status // 查看git仓库状态

8、推送提交到GitHub
git push -u origin master

============
Git的中文支持
============
解决中文支持方法：
配置core.quotepath 属性为 false， 就可以解决中文乱码问题
git config --global core.quotepath false
